# Kenny AI/ML Portfolio 🤖

A collection of AI and Machine Learning projects built with Python.

## Projects

| Project | Description | Technology |
|---------|-------------|------------|
| 🏠 House Price Predictor | Predicts house prices based on size | Linear Regression |
| 🧹 Data Cleaning | Cleans and transforms messy datasets | Pandas |
| 👥 Customer Churn | Predicts if a customer will leave | Random Forest |
| 🤖 AI Chatbot | Conversational AI powered by Claude | Anthropic API |
| 🧠 Neural Network | Classifies customers using deep learning | TensorFlow/Keras |
| 🚢 Titanic Survival | Predicts Titanic survival from real data | Kaggle/Random Forest |
| 💬 Sentiment Analyzer | Analyzes positive/negative sentiment | NLP/TextBlob |
| 🎬 Movie Recommender | Recommends movies based on user taste | Cosine Similarity |
| 📊 Portfolio Dashboard | Interactive data visualization dashboard | Dash/Plotly |
| 🔢 Digit Recognizer | Recognizes handwritten digits with 97.6% accuracy | Computer Vision/CNN |
| 📈 Stock Predictor | Predicts Apple stock prices using time series | LSTM Neural Network |

## Tech Stack
- **Languages:** Python
- **ML Libraries:** Scikit-learn, TensorFlow, Keras
- **Data:** Pandas, NumPy, Matplotlib, Seaborn
- **NLP:** NLTK, TextBlob
- **Dashboard:** Dash, Plotly
- **API:** Anthropic Claude API
- **Data Source:** Kaggle, Yahoo Finance
